# 💰 Debt Manager

A web application for managing personal debt, calculating interest, and planning repayments.

## Features

- ✅ Add debts
- ✅ Delete debts
- ✅ Automatically saves data
- ✅ Loads saved debts
- 🚧 Edit debts (coming soon)
- 🚧 Repayment simulator
- 🚧 Avalanche & Snowball strategies
- 🚧 Interest forecasting
- 🚧 Credit utilisation tracking

- HTML
- CSS
- JavaScript
- Git
- GitHub
